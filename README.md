# 基于完全vibe coding的第一个github项目————可用于edge以及chrome浏览器的文字及视频字幕翻译插件
## 支持基于gemini api的网页文字翻译和英文视频生成双语字幕
## 核心功能

### 1. 一键网页翻译
- 遍历页面 DOM，收集英文文本节点
- 分批发送给 Gemini API 翻译，避免超限
- 原地替换文本，支持一键恢复原文
- 翻译缓存避免重复调用

### 2. 视频字幕
- 检测页面 `<video>` 元素
- 通过 `tabCapture` 捕获标签页音频
- Web Speech API 实时语音识别
- Gemini 翻译为中文，叠加中英双语字幕
- 字幕样式：白色英文 + 黄色中文，半透明背景

## 使用方法

### 安装步骤
1. 打开 Chrome/Edge → 地址栏输入 `chrome://extensions/`（或 `edge://extensions/`）
2. 开启右上角**「开发者模式」**
3. 点击**「加载已解压的扩展程序」**
4. 选择插件所在的文件夹

### 配置 API Key
1. 点击插件图标 → 右上角齿轮图标（或右键插件图标 → 选项）
2. 在 [Google AI Studio](https://aistudio.google.com/app/apikey) 获取 API Key
3. 粘贴到输入框 → 点击保存

### 翻译网页
1. 访问英文网页
2. 点击插件图标 → 点击**「翻译页面」**
3. 等待翻译完成，文字将原地替换为中文
4. 再次点击切换为**「恢复原文」**

### 视频字幕
1. 访问含有英文视频的页面
2. 播放视频
3. 点击插件图标 → 点击**「开启字幕」**
4. 字幕将在视频底部实时显示
